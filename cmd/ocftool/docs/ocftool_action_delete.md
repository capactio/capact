## ocftool action delete

Deletes the Action

```
ocftool action delete ACTION_NAME [ACTION_NAME...] [flags]
```

### Options

```
  -h, --help               help for delete
  -n, --namespace string   Kubernetes namespace where the Action was created (default "default")
```
