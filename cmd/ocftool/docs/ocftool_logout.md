## ocftool logout

Log out from a Gateway server

```
ocftool logout [SERVER] [flags]
```

### Examples

```
ocftool logout
# => select what server to log out of via a prompt

ocftool logout localhost:8080
# => log out of specified server

```

### Options

```
  -h, --help   help for logout
```
